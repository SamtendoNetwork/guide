# Common Issues & Fixes

Having an issue with Samtendo Network? Here are some of the most common ones!

## Error codes
Use the command *e-* followed by your error code (e.g. e-160-0103) in the **#commands** channel in the [Samtendo Network Discord server](https://discord.gg/jDKvdtVyzK).

If you're banned from our Discord server, email help@samtendo.net to appeal.
