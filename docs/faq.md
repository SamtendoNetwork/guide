# Frequently Asked Questions

## Does Samtendo Network have any risks?

As Nintendo Network has closed down, it is impossible for you or your console to be banned. As long as you follow this guide carefully, bricking will be *almost* impossible too. However, it's good to know that changes made by Electrode (the plugin used to connect to Samtendo Network) does not make permanent changes, and everything is done in-memory.

## What consoles are supported?

Currently, only the Wii U is supported. That includes both the 8GB (white) and 32GB (black) models. Any region should work, but please ensure your console is updated to the latest version (5.5.6 in North America, 5.5.5 elsewhere).
