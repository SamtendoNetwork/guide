# Getting Started

Before starting this guide, we need to check if custom firmware (CFW) is already installed on your console, alongside other potentially conflicting plugins.

### Pre-check

Please follow the instructions for your console.

| Image | Instructions |
| ----- | ------------ |
| ![Wii U](/assets/img/guide/WhiteU.png) | Carry on to Section I |
| ![3DS](/assets/img/guide/3ds.png) ![2DS](/assets/img/guide/2ds.png) | *3DS/2DS is not supported yet by this guide* |
| ![Wii](/assets/img/guide/Wii-Console.png) | *Wii is not supported yet by this guide* |

### Section I - CFW Check

1. Power off your Wii U
1. Ensure the SD Card is inserted into your console
1. Power on your Wii U and wait for it to turn on
1. Once at the user select screen, press **L + D-Pad Down + Select** at the same time
1. If the Aroma plugin manager pops up, you may proceed to the next section.

::: warning

If you do not see the Aroma plugin manager or you see another CFW's menu, **STOP** - you do not have Aroma installed. Continue from [here](https://wiiu.hacks.guide) and return once you have finished.

:::

### Section II - Version Check

1. Launch System Settings
1. Look at the Wii U GamePad. In the top right, you will see your console's version (e.g. 5.5.5 E).

If your system is a North American one, you should be on 5.5.6 U. Otherwise, your version should be 5.5.5. If it's not on the latest version, please perform a system update, then continue.

If all is good, you're ready to continue!
