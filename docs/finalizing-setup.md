# Finalizing Setup

With your SD Card back in your Wii U, power it back on. Upon reaching the Wii U Menu you should see the following popups in the top left:

![Electrode popups](assets/img/guide/popups.png)

::: danger

If you do not see these exact popups, you have likely misplaced a file. Please go back and review the instructions carefully. Otherwise, ask for help in the [Samtendo Network Discord server](https://discord.gg/jDKvdtVyzK).

:::

Once at the user selection screen, choose *Add New User* in the top right. From this point on your console will guide you through creating a Samtendo Network ID. **You have now finished installing Samtendo Network on your Wii U!**

## What next?

::: tip

We highly recommend joining the [Samtendo Network Discord server](https://discord.gg/jDKvdtVyzK), as you can stay up to date on updates, chat with other Wii U fans and get quick support in the event you need it!
